What is STRK20

Starknet-native privacy layer for ERC-20 tokens. One canonical multi-asset pool.

One pool, all tokens
Single multi-asset pool for any ERC-20 on Starknet. No fragmented liquidity.
Note discovery
Wallets scan user channels, not the whole pool. Fast fund discovery without compromising privacy.
Anonymous DeFi
Open notes enable anonymous swaps against existing liquidity. Who traded is hidden, not the trade itself.
Selective disclosure
Viewing keys let an authorized entity audit one user without affecting others.
Same account model
No second wallet, no separate chain. Privacy inside the existing Starknet ecosystem.
Cairo + STARKs
No trusted setup, in-protocol proof verification, native account abstraction.
Tradeoffs Practical privacy, not maximalist. Deposits/withdrawals are public. Anonymous swaps hide who traded, not amounts. Selective disclosure adds an audit path for regulated use.
